# template-vite-react-tailwind-jit

- **template of:** vite + react
- **tailwindcss:** v2.2.7

## Started
with npm:
```bash
npm install
```
with yarn:
```bash
yarn
```

---
## Develop
with npm:
```bash
npm run dev
```
with yarn:
```bash
yarn dev
```

## Develop with docker
```bash
docker build -t short_url -f ./docker/dockerfile .  docker build -t short_url -f ./docker/dockerfile .
docker run -p 3000:3000 short_url
```

---
## Build
with npm:
```bash
npm run build
```
with yarn:
```bash
yarn build
```
